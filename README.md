# vigenere CLI

A CLI for vigenere.

## Customizing your CLI

Check out the documentation at https://github.com/infinitered/gluegun/tree/master/docs.

## Setup
`npm link` or `yarn link`

## Commands

```shell
$ vigenere
$ vigenere -h
$ vigenere -v
$ vigenere decrypt <file_path> <common_char> --language=(en|pt)
```
